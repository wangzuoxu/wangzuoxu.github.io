# Academic Homepage Starter

This folder contains a static personal academic homepage that can be published with GitHub Pages.

## How to use

1. Replace `Your Name`, `Your Institution`, email, and profile links in `index.html`.
2. Edit the publications, projects, and news sections with your actual content.
3. Optionally replace the placeholder profile circle with a real portrait image.
4. Commit the files and push them to GitHub.

## GitHub Pages setup

1. Open the repository on GitHub.
2. Go to `Settings` > `Pages`.
3. Set the source to the branch and folder that contains this site.
4. Save and wait for the site URL to be generated.

## Local preview

Open `index.html` directly in a browser, or serve the folder with any static server.
